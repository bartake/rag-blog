16. Advanced RAG Patterns

Agentic RAG, multi-hop, knowledge graphs, temporal, multi-modal

Agentic RAG

Agents combine retrieval with tool use (search, calculator, code execution). The LLM decides when to retrieve, when to call tools, and how to synthesize. Enables dynamic, multi-step reasoning beyond static context injection.

Multi-Hop Reasoning

Complex questions require multiple retrievals. Answer A leads to retrieval B. Iterative retrieval—use initial results to form follow-up queries—enables chain-of-thought style reasoning over documents.

Knowledge Graph + RAG Hybrid

Combine vector search with graph traversal. Embeddings find candidate nodes; graph edges provide structure (relations, hierarchy). Use for "entity + relation" queries and explainability.

Temporal RAG

Time-aware retrieval: filter by date, retrieve latest docs, or weight recency. Critical for news, policies, and fast-changing domains.

Multi-Modal RAG

Retrieve and generate across text and images. Use vision-language models (VLMs) for image embeddings; joint embedding spaces for text+image similarity. Enables visual QA and document understanding.
