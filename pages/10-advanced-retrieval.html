10. Advanced Retrieval Techniques

Re-ranking, parent-child retrieval, fusion, and more

Techniques
Re-ranking (Cross-encoders)
Score (query, passage) together. Top-20→top-5 refinement. Slower, more accurate.
Parent-Child Retrieval
Children for retrieval precision; return parent for context. Granularity + coherence.
Recursive Retrieval
Initial chunks → identify gaps → follow-up queries. Iterative until sufficient.
Context Compression
LLM compresses passages before injection. Fewer tokens, key info preserved.
RRF (Reciprocal Rank Fusion)
Merge vector + BM25. RRF(k)=Σ 1/(k+rank). No score calibration needed.
