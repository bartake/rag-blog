1. Foundations: AI & NLP Basics

Core concepts you need before diving into RAG

What is Generative AI
Models that create new content (text, images, code) from training patterns. For RAG: LLMs—autoregressive models predicting next tokens for coherent generation.
LLM Fundamentals
Tokens
~4 chars or 0.75 words each. Affects cost, latency. "RAG" ≈ 4–5 tokens.
Context Window
Max tokens the model sees (4k–128k+). Retrieved passages + query must fit.
Temperature
0–0.3 = factual, 0.7–1 = creative. Use low for RAG to reduce hallucination.
Embeddings
Dense vector arrays (768–1536 dims) capturing semantic meaning. Similar concepts cluster. RAG uses them for semantic similarity search, not keywords.
Transformers
Self-attention weighs token relationships. Encoder → embeddings; Decoder → text. LLMs are decoder-only. Handle variable-length, parallel processing.
Prompt Engineering
Key techniques
Clear instructions, few-shot examples, role-setting, delimiters. Tell model: use only context; say "I don't know" when absent.
