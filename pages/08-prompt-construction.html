8. Prompt Construction

Context window, templates, instructions, guardrails

Best Practices
Context Window
Stay within limit (128K). Prioritize top chunks. Truncate/summarize if needed.
Templates
System + context + query. Use ---CONTEXT--- delimiters.
Instruction vs Context
Keep "how to answer" separate from "retrieved facts." Improves faithfulness.
Guardrails
"Answer only from context. Say 'I don't know' if unsure." Reduces hallucination.
