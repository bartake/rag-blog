13. Production Architecture

End-to-end system design for real-world deployments

Architecture
System Design
Ingest (batch + real-time), retrieval, LLM gateway, API. Document flow, failures, SLAs.
API & Orchestration
Single RAG API. Query→embed→search→prompt→generate. Async where possible.
Async Pipelines
Kafka, SQS for ingest. Decouple indexing from query path.
Failure & Retries
Retries, fallbacks. Graceful degradation (e.g., keyword-only if vector DB down).
Observability
Logs, OpenTelemetry traces. Dashboards: latency, cost, error rate.
