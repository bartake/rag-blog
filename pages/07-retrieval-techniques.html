7. Retrieval Techniques

Top-K, thresholds, hybrid search, query expansion, multi-query

Core Techniques
Top-K Retrieval
Return K most similar. K=5–10 common. Too small misses; too large adds noise.
Similarity Thresholds
Cosine > 0.7 cutoff. Filters weak matches; can over-filter if noisy.
Hybrid Search
BM25 + vector. RRF to merge. Covers exact & semantic.
Query Expansion
Synonyms, paraphrases. Retrieve for each, merge. Better recall.
Multi-Query
LLM creates variants, retrieve each, deduplicate. Robust to phrasing.
